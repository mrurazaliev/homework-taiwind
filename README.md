#  YtYt — Programming School Website

Modern responsive landing page for an online programming school.

The website was created using HTML and Tailwind CSS with adaptive design for desktop and mobile devices.

---

#  About The Project

YtYt is an educational platform website that introduces users to programming courses, learning methods, technologies, pricing, and the educational process.

The website contains modern UI components, responsive layouts, and structured content blocks for comfortable user experience on all devices.
